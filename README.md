# Sistema POS - Inventario (WinForms)

Aplicación de escritorio desarrollada en C# con .NET WinForms, utilizada en un entorno real para la gestión de inventario en un negocio tipo POS.

El enfoque principal del proyecto es la funcionalidad, rapidez y usabilidad en operaciones diarias, priorizando soluciones prácticas sobre complejidad innecesaria.

---

## Características principales

* Gestión de inventario
* Búsqueda dinámica de productos
* Paginación para grandes volúmenes de datos
* Edición manual con control de cambios
* Resaltado visual de filas modificadas
* Sistema de Undo (Ctrl + Z)
* Ordenamiento de columnas
* Exportación a Excel
* Importación desde Excel

---

## Enfoque del desarrollo

Este proyecto fue construido con un enfoque práctico y orientado a uso real, priorizando:

* Rendimiento con grandes volúmenes de datos
* Fluidez en la edición masiva de información
* Minimización de errores en operaciones manuales
* Experiencia de usuario clara y rápida

### Soluciones implementadas:

* Manejo de datos en memoria (cache + vista paginada)
* `VirtualMode` en DataGridView para rendimiento
* Seguimiento manual de cambios en edición
* Sistema de deshacer (Undo)

---

## Tecnologías utilizadas

* C#
* .NET (WinForms)
* Entity Framework
* SQLite
